[[type-Result]]
=== union Result[T]

Fallible value that is either a Success with a value or a structured Error.

Expected, recoverable failures should be represented in return types as
`Result[T]` or `Effect[Result[T]]` instead of Java-style checked
exceptions.

==== Variants

* xref:#type-Error[Error]
* xref:#type-Success[Success][T]

==== Functions

===== public Result[T].`|*`(flatmap: T=>Result[Y]): xref:#type-Result[Result][Y]

Applies `flatmap` to a successful value; equivalent to `this.flat_map(flatmap)`.


===== public Result[T].`|>`(reduce: xref:../../capy/collection/Tuple.adoc#type-Tuple[Tuple][T=>R, Error=>R]): R

Reduces a Result by applying the first function to Success or the second function to Error.


===== public Result[T].`|`(map: T=>Y): xref:#type-Result[Result][Y]

Transforms a successful value using `map`; equivalent to `this.map(map)`.


===== public Result[T].error_message(): xref:../../capy/lang/String.adoc#type-String[String]

Returns the error message, or an empty string when this Result is successful.


===== public Result[T].flat_map(flatmap: T=>Result[Y]): xref:#type-Result[Result][Y]

Applies `flatmap` to a successful value, or propagates an Error unchanged.


===== public Result[T].map(map: T=>Y): xref:#type-Result[Result][Y]

Transforms a successful value using `map`, or propagates an Error unchanged.


===== public Result[T].map_error(map: Error=>Error): xref:#type-Result[Result][T]

Maps an Error while preserving successful values.


===== public Result[T].or(r: xref:#type-Result[Result][T]): xref:#type-Result[Result][T]

Returns this Result when successful, or `r` when this Result is an Error.


===== public Result[T].or_else(v: T): T

Returns the successful value, or `v` when this Result is an Error.


===== public Result[T].recover(recover: Error=>T): T

Recovers an Error into a plain value.


===== public Result[T].recover_result(recover: Error=>Result[T]): xref:#type-Result[Result][T]

Recovers an Error into another Result.


===== public Result[T].reduce(on_success: T=>R, on_error: Error=>R): R

Reduces a Result by applying `on_success` to Success or `on_error` to Error.


===== public Result[T].reduce_left(on_success: T=>R, on_error: Error=>R): R

Reduces a Result by applying `on_success` to Success or `on_error` to Error.
