= Module List

== Types

[[type-List]]
=== data List[T]

Native ordered collection type backed by the runtime List implementation.
List literals use square brackets: [1, 2, 3].

Example:
```cfun
let values = [1, 2, 3]
values + 4
```
Lists are immutable: `values + 4` returns a new list and leaves `values`
unchanged.

Returns true when every value satisfies `predicate`.

===== public List[T].any(predicate: T=>bool): bool

Returns true when at least one value satisfies `predicate`.


===== public List[T].contains(value: T): bool

Returns true when this List contains the given value.


===== public List[T].filter(predicate: T=>bool): xref:../../capy/collection/Seq.adoc#type-Seq[Seq][T]

Returns a sequence containing only values that satisfy `predicate`.

Example:
```cfun
([1, 2, 3, 4] |- value => value > 2).as_list()
```
Returns `[3, 4]`.


===== public List[T].flat_map(map: T=>List[Y]): xref:../../capy/collection/Seq.adoc#type-Seq[Seq][Y]

Applies `map` to each value and concatenates the resulting lists.

Example:
```cfun
([1, 2] |* value => [value, value * 10]).as_list()
```
Returns `[1, 10, 2, 20]`.


===== public List[T].fold(reducer: (T,T)=>T): xref:../../capy/lang/Option.adoc#type-Option[Option][T]

Combines values from left to right using the first value as the accumulator.

Empty lists return `None`. Non-empty lists return `Some` with the folded value.


===== public List[T].get(idx: xref:../../capy/collection/CollectionTypes.adoc#type-index[index]): xref:../../capy/lang/Option.adoc#type-Option[Option][T]

Returns the value at idx using the same bounds and negative-index rules as list[idx].

Example: `[10, 20, 30].get(-1)` returns `Some { value: 30 }`.


===== public List[T].get(from: int, to: int): xref:#type-List[List][T]

Returns the values in the half-open range [from, to) using the same bounds and negative-index rules as list[from:to].

Example: `[10, 20, 30, 40].get(1, 3)` returns `[20, 30]`.


===== public List[T].is_empty(): bool

Returns true when this List has no values.


===== public List[T].map(map: T=>Y): xref:../../capy/collection/Seq.adoc#type-Seq[Seq][Y]

Transforms each value using `map`.

`map` returns a lazy `Seq`, so call `as_list()` when a materialized list is
needed.

Example:
```cfun
([1, 2, 3] | value => value * 2).as_list()
```
Returns `[2, 4, 6]`.
