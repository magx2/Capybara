[[type-Dict]]
=== data Dict[T]

Native String-keyed map type backed by the runtime Dict implementation.
Dict literals use braces with String keys: {"one": 1, "two": 2}.

Example:
```cfun
let base = { "name": "Ada" }
base + { "role": "admin" }
```
Dictionaries are immutable: adding or removing entries returns a new `Dict`.

